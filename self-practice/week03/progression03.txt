week03 ให้ตรวจสอบว่าใน sentence มีแต่ละคำเท่าไหร่ โดย ignore case
เริ่มแรกเราตั้ง function getFreqOfWords(sentence) โดยที่ sentence ไม่เป็นค่า null, undefined เช็คโดยใช้ if ถ้าหากไม่เข้าเงื่อนไข จะ return undefined และหากเข้าเงื่อนไขจะเข้าสู่ขั้นตอนการนับจำนวนคำ แต่จะเริ่มโดยการ สร้าง wordCount เป็น object เปล่า เพื่อเก็บผลลัพธ์ และ wordArray เพื่อให้ทุกตัวในประโยคเป็นตัวเล็ก ใช้ toLowerCase และใช้ split เพื่อให้แยกแต่ละคำในประโยค(เปลี่ยน string เป็น array) จากนั้นให้ลูปใน wordArray โดยใช้  word for...of และใช้ if ช่วยเพื่อแยกเงื่อนไข หาก wordCount[word](มีคำเหมือนกัน) ให้ wordCount[word]++(เพิ่มขึ้นทีละ 1) ถ้าหากไม่ใช่ ให้ wordCount[word] คงค่าเป็น 1
